Unity Bakes: International Bakery

Game Design
Time 5 Months (Feb-Jun 2024)
Team Members Andrea Borsato, Filippo Randon, Zhuoyue Song, Giulia JiangXian Zhu
Role Game Design / Visual Design / Interaction Design
Course Politecnico di Milano - Virtual and Physical Prototyping
Report outbound

Project Overview

UnityBakes is an international bakery where players can learn about pastries from different countries. It's a fun place for cultural exchange and sharing.

Storyboard


1. Explore different stands to learn about breads from various countries.


2. Go to the cashier and answer a question.


3. Head to the sharing table.


4. Teleport to the Eden of Bread.


5. Touch the Golden Bread.


6. Explosion: Game over.
